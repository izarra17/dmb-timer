# ДМБ Таймер — личное PWA для телефона

## Как открыть на телефоне

### Способ 1: через Wi‑Fi с компьютера (рекомендуется)

1. На компьютере откройте терминал в этой папке.
2. Запустите сервер:
   ```
   python -m http.server 8080
   ```
   Если Python не установлен:
   ```
   npx --yes serve -p 8080
   ```
3. Узнайте IP компьютера (`ipconfig` в Windows, строка IPv4).
4. На телефоне (тот же Wi‑Fi) откройте Chrome:
   ```
   http://ВАШ_IP:8080
   ```
5. Меню Chrome (⋮) → **«Добавить на главный экран»**.

### Способ 2: скопировать папку на телефон

1. Скопируйте папку `календарь` на телефон.
2. Откройте `index.html` в Chrome.
3. Настройки сохранятся в браузере.

## Первый запуск

1. Нажмите **настройки** (шестерёнка) слева вверху.
2. Введите имя, дату призыва и срок (12 или 18 месяцев).
3. Нажмите **Сохранить**.

## Функции

- Таймер в реальном времени
- Процент службы и полоска прогресса
- Праздники внизу экрана
- Кнопка «глаз» — скрывает цифры
- Данные только на вашем телефоне
